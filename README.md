# StarScraper 

A custom new tab page, that can reach the stars!

<img width="1896" height="901" alt="260719_22h51m08s_screenshot" src="https://github.com/user-attachments/assets/debb0d69-0086-4715-aaf1-0cc2ee95825d" />

## Features

- Customizable search bar
- Favorites
- Data from [Stardance](https://stardance.hackclub.com)
- Local weather information

## Hosting

[This page](https://starscraper.kyus.hu) is written in pure HTML/CSS/JS, no fancy frameworks needed, and it's hosted on Github Pages.

## Notes

This project was made apart of Hack Club's Stardance Challenge, specifically the "Give Your Website a Pulse" mission.
